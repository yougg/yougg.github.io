配置GoLand集成开发环境

2017年11月30日 16时38分52秒 发布在 Document

安装必备工具

下载安装以下软件，确保执行命令的路径都已添加到系统环境变量PATH中

Golang

  • 安装向导
    1. 使用默认安装路径

Notepad++

  • 安装向导
    1. 使用默认安装路径
    2. 勾选默认安装组件
    3. 勾选创建桌面快捷方式

Graphviz

  • 安装向导
    1. 使用默认安装路径
    2. 安装后需要手工添加C:\Program Files (x86)\Graphviz2.38\bin到环境变量PATH中
    3. 添加环境变量后，打开CMD执行dot -V查看版本

Git

  • 安装向导
    1. 使用默认安装路径
    2. 勾选所有安装组件
    3. 使用Vim作为Git默认编辑器
    4. 在Windows命令中使用Git和可选的Unix工具
    5. 使用OpenSSL库
    6. Checkout as-is, commit Unix-style line endings
    7. 使用MinTTY终端模拟器
    8. 启用文件系统缓存,启用Git凭证管理,启用符号链接

GoLand EAP版本 正式版

  • 安装向导

    1. 自定义安装路径
    2. 勾选创建桌面快捷方式

  • 配置向导

    • 启动GoLand，选择不导入配置，同意用户协议
    • 点击右下角Configure -> Settings进入设置
    • 选择Go -> GOROOT，下拉选择Go SDK

    • 选择Go -> GOPATH，去除勾选的两项：

      Use GOPATH that’s defined in system environment
      Index entrie GOPATH

    • 选择Go -> Imports，排序类型下拉选择goimports，并勾选下面3个分组选项

      Group stdlib imports
      　 Move all stdlib imports in a single group
      Move all imports in a single declaration

    • 选择Appearance & Behavior -> Appearance

      • Theme下拉选择Darcula

      • Override default fonts by(not recommenended):

        Name: 推荐使用Source Code Pro，Size: 推荐14号

    • 选择Appearance & Behavior -> System Settings -> HTTP Proxy设置自己的代理

    • 选择Keymap，点击方案列表下拉框右边设置选择Duplicate...然后重命名

      • 点击展开Main menu -> Code -> Completion
        • 右键点击Basic，移除Ctrl+空格
        • 右键点击SmartType，移除Ctrl+Shift+空格
        • 右键点击Cyclic Expand Word，移除Alt+/
        • 右键点击Cyclic Expand Word (Backward)，移除Alt+Shift+/
        • 右键点击Basic，点击Add Keyboard Shortcut，按快捷键Alt+/然后点击OK
        • 右键点击SmartType，点击Add Keyboard Shortcut，按快捷键Alt+Shift+/然后点击OK
        • 右键点击Cyclic Expand Word，点击Add Keyboard Shortcut，按快捷键Ctrl+空格然后点击OK
        • 右键点击Cyclic Expand Word (Backward)，点击Add Keyboard Shortcut，按快捷键Ctrl+Shift+空格然后点击OK

    • 选择Editor -> General -> Code Folding，去除勾选Imports

      Imports

    • 选择Plugins

      • 勾选与勾除下列默认的插件

        Plugin Checked
        Copyright
        CSS Support
        Database Tools and SQL
        EditorConfig
        File Watchers
        Git Integration
        GitHub
        Go
        Go IDE
        HTML Tools
        IntelliLang
        JavaScript Debugger
        JavaScript Intention Power Pack
        JavaScript Support
        Mercurial Integration
        QuirksMode
        Refactor-X
        REST Client
        Terminal
        TextMate bundles support
        tslint
        XPathView + XSLT Support
        YAML

      • 点击Browse repositories...，搜索并安装以下插件

        • .ignore
        • BashSupport
        • Markdown support
        • PlantUML
        • IDEA Mind Map
        • Ini4Idea
        • Key Promoter X

    • 选择Tools -> Terminal，在Shell path中填入

      "C:\Program Files\Git\bin\bash.exe" --login -i

    • 点击OK，然后点击Restart重启

      • 点击New Project
        • Location选择新建一个目录，如：D:\Workspace\20180208
        • Sdk点击下拉选择
        • 点击Create创建工程
      • 点击Open Project
        • 选择目录，如：D:\workspace\Proj

    • 按快捷键Ctrl+Shift+A -> 输入Toolbar，点击OFF变为ON

    • 按快捷键Ctrl+Shift+A -> 输入Navigation Bar，点击ON变为OFF

    • 按快捷键Ctrl+Shift+A -> 输入Tool Buttons，点击OFF变为ON

    • 按快捷键Ctrl+Shift+A -> 输入Autoscroll from Source，点击OFF变为ON

    • 按快捷键Ctrl+Shift+A -> 输入Autoscroll to Source，点击OFF变为ON

    • 按快捷键Ctrl+Shift+A -> 输入File Types回车

      在Recognized File Types中选择Ini
      在下方Registered Patterns点击+
      输入*.properties点击OK
      再点击+，输入*.conf点击OK

    • 点击File菜单，选择Settings -> Other Settings -> PlantUML

      Graphviz dot executable 点击Browse…，选择：
      C:/Program Files (x86)/Graphviz2.38/bin/dot.exe

GoLand配置一个Project中支持多个Module

  • 使用GoLand分别打开多个项目，File -> Open...选择项目目录
  • 关闭打开的项目，保留需要支持多个Module的项目
  • 用文本编辑器打开各项目根目录下的Module配置文件 .idea/modules.xml

  • 复制module标签内容到保留的Module的配置文件中

    <module fileurl="file://$PROJECT_DIR$/.idea/proj.iml" filepath="$PROJECT_DIR$/.idea/proj.iml" />

  • 修改复制的module标签中文件路径，使用相对路径或者绝对路径

    <?xml version="1.0" encoding="UTF-8"?>
<project version="4">
    <component name="ProjectModuleManager">
        <modules>
            <module fileurl="file://$PROJECT_DIR$/.idea/proj.iml" filepath="$PROJECT_DIR$/.idea/proj.iml" />
            <module fileurl="file://$PROJECT_DIR$/../module0/.idea/module0.iml" filepath="$PROJECT_DIR$/../module0/.idea/module0.iml" />
            <module fileurl="file://$PROJECT_DIR$/../module1/.idea/module1.iml" filepath="$PROJECT_DIR$/../module1/.idea/module1.iml" />
        </modules>
    </component>
</project>

  • 在GoLand中保留的项目里面刷新即可显示多个Module
