30.编译并运行下面的程序，运行结果为 (初级题3分)

public class T1
{
    public static void main (String[] args)
    {
        T1 a=new T1();
        a.method(8);
        a.method(1.2f);
    }
    void method(float i)
    {
        System.out.println("float: "+i);
    }
    void method(long i)
    {
        System.out.println("long: "+i);
    }
}
B.程序可以编译运行，输出结果为： long: 8f loat: 1.2
D.程序中定义的两个参数不同的method方法称为方法重栽
